\# Financial Return Modeling



This project compares traditional machine learning and deep learning models for predicting Microsoft stock returns.



\## Project Overview



The project uses historical MSFT stock data from Yahoo Finance and applies feature engineering, scaling, linear regression, and an LSTM neural network to predict next-day stock returns.



\## Models Used



\- Linear Regression

\- Long Short-Term Memory Neural Network



\## Features



The model uses:



\- Daily return

\- 5-day simple moving average

\- 10-day simple moving average

\- 5-day volatility

\- Trading volume
